## Instalación e iniciar el servidor
```sh
git clone https://github.com/JuanVqz/rails-spa.git
cd rails-spa
bundle install

rails db:create
rails db:migrate

rails serve
```

## Versiones
 * Ruby 2.5.3
 * Rails 5.2.2
